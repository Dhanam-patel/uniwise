# Uniwise

This is an independent React application created using Vite, representing the Uniwise front-end experience.

## Getting Started

To run the application locally:

### Prerequisites

- Node.js installed

### Installation

```bash
npm install
```

### Development Server

Start the local development server:

```bash
npm run dev
```

The application will be accessible at `http://localhost:5173`.

### Building for Production

Compile optimized production build:

```bash
npm run build
```

The built website will be hosted in the `/dist` directory.

## Project Structure

This project follows a standard scalable React structure:

- `src/components/`: Reusable UI elements (Buttons, Navbar, Layouts).
- `src/pages/`: Major application views and screens.
- `src/lib/`: Global utilities, shadcn/ui components, and helper functions.
- `src/hooks/`: React hooks for shared logic.
- `src/assets/`: Static image, SVG, and branding assets.
